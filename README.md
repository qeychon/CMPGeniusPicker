## Installation

CMPGeniusPicker is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CMPGeniusPicker', :git => 'git@github.com:qeychon/CMPGeniusPicker.git'
```
